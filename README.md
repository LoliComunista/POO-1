📖 Sobre o Projeto

Este software foi desenvolvido utilizando os princípios da Programação Orientada a Objetos (POO), aplicando conceitos fundamentais como encapsulamento, herança e polimorfismo.

O sistema tem como objetivo organizar e facilitar o gerenciamento de dados, oferecendo uma solução simples, estruturada e eficiente para o usuário.

🚀 Funcionalidades
Cadastro de informações
Listagem de dados
Atualização de registros
Exclusão de dados
Interface simples e intuitiva
🛠️ Tecnologias Utilizadas
Linguagem: Java
Paradigma: Programação Orientada a Objetos
Interface: (Swing / Console — ajuste conforme seu projeto)
🧠 Conceitos Aplicados
Encapsulamento: Proteção dos dados através de métodos getters e setters
Herança: Reutilização de código entre classes
Polimorfismo: Flexibilidade na utilização de métodos
▶️ Como Executar
Clone o repositório
Abra o projeto em uma IDE (ex: NetBeans, Eclipse ou IntelliJ)
Execute a classe principal
📌 Objetivo

O projeto foi desenvolvido com foco educacional, visando a prática dos conceitos de POO e a construção de uma aplicação funcional.
